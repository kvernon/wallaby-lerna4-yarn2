## testing with coverage

There are new pieces added to the jest.unit.preset, which omit certain files for coverage.

in here it is:

```javascript
{
  collectCoverageFrom: [
    '!**/tests/integration/**',
    '!**/**/.eslintrc.js',
    '!**/**/jest.config.js',
    '!**/**/jest.integration.config.js'
  ]
}
```
